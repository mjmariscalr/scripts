# Scripts

Repositorio personal donde almaceno scripts simples relacionados con automatización, pentesting, administración de sistemas y aprendizaje de ciberseguridad.

## Tecnologías utilizadas

Dependiendo del script:

- Bash
- Python
- Herramientas Linux y de redes

## Contenido

- [**SMBfuzz:**](https://github.com/mjmariscalr/scripts/tree/main/smbfuzz) enumera recursos compartidos SMB utilizando autenticación anónima.

## Aviso

Todos los scripts de este repositorio están destinados únicamente a fines educativos y de laboratorio.  
No deben utilizarse contra sistemas sin autorización explícita.
